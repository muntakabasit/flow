┌─ ANIMATION & PHYSICS LAYER ───────────────────────────────────────────────┐
│                                                                              │
│  OrbView_ModeB 🎯                                                           │
│  ├── 3-Layer Architecture                                                  │
│  │   ├── Core: Gradient circle, scales with state                         │
│  │   ├── Glow: Blurred halo, opacity/radius driven                        │
│  │   └── Halo Ring: Thin rotating ring (translating state only)           │
│  │                                                                          │
│  ├── Peak-Level Smoothing Algorithm                                        │
│  │   └── smoothed = smoothed * 0.85 + raw * 0.15                          │
│  │       (Exponential moving average, no jitter)                          │
│  │                                                                          │
│  ├── State Animations                                                      │
│  │   ├── IDLE: Breathing ±1.5% scale, 3.2s cycle, easeInOut             │
│  │   ├── LISTENING: Peak-driven, scale 1.02-1.12, responsive             │
│  │   ├── TRANSLATING: Stable scale 1.04, halo rotation 6s                │
│  │   ├── SPEAKING: Ripples outward (2-3 rings, 1.2s each)               │
│  │   └── OFFLINE: Desaturated, no animation                              │
│  │                                                                          │
│  ├── Spring Animations (State Transitions)                                │
│  │   └── response: 0.35-0.55, dampingFraction: 0.85-0.95                │
│  │       (Never snappy, always smooth)                                    │

│  ├── Audio Level → Visuals Mapping                                        │

│  │   ├── Glow radius: 8 + smoothedPeak * 18                              │
│  │   └── Glow opacity: 0.18 + smoothedPeak * 0.25                        │

┌─ DATA FLOW: FROM AUDIO TO VISUAL ─────────────────────────────────────────┐
│                                                                              │
│  1. User Speaks                                                            │
│     └── Microphone captures 16kHz PCM                                      │
│                                                                              │
│  2. Audio Processing                                                       │
│     ├── AudioService analyzes energy                                       │
│     ├── Computes peak level (0-1 range)                                    │
│     └── Fires onPeakLevel(Float) callback (0-60 times/sec)               │
│                                                                              │
│  3. Peak Level → State                                                     │
│     ├── FlowCoordinator receives peak in onPeakLevel callback             │
│     ├── Throttle to 30fps (CFAbsoluteTimeGetCurrent check)               │
│     └── appState.audioLevel = Float(level) [Published, all views observe] │
│                                                                              │
│  4. OrbView_ModeB Responds                                                │
│     ├── @ObservedObject appState                                          │
│     ├── onChange(of: appState.audioLevel)                                │
│     │   └── updateSmoothedPeak(newLevel)                                 │
│     │       └── smoothed = smoothed * 0.85 + raw * 0.15                  │
│     │           └── withAnimation(.spring(...))                           │
│     │               └── Apply spring to smoothed value                     │
│     │                                                                      │
│     └── Map smoothed to visuals in updateOrbAnimation()                   │
│         ├── core scale = 1.02 + smoothed * 0.10                          │
│         ├── glow radius = 8 + smoothed * 18                              │
│         ├── glow opacity = 0.18 + smoothed * 0.25                        │
│         └── Update happens every 0.016s (60 FPS)                         │
│                                                                              │
│  5. Result: Smooth, Responsive Orb                                        │
│     └── No jitter, never snappy, feels premium                           │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘


┌─ TURN BOUNDARY LOGIC ─────────────────────────────────────────────────────┐
│                                                                              │
│  Speech Active                                                             │
│  └── appState.transition(.listening)                                       │
│                                                                              │
│  User Pauses (speechStopped event)                                        │
│  └── TurnSmoothingManager.onSpeechStopped(timestamp)                      │
│      └── Schedule silence evaluation (100ms delay)                        │
│                                                                              │
│  Evaluate Silence Duration                                                │
│  ├── Measure time since speech stopped                                    │
│  │                                                                          │
│  ├── < 0.4s: Micro-pause (likely mid-sentence)                           │
│  │   └── IGNORE (continue current turn)                                   │
│  │                                                                          │
│  ├── 0.4-0.7s: Maybe pause (prepare, don't finalize yet)                 │
│  │   └── Set state to "waiting for finalize"                             │
│  │                                                                          │
│  ├── 0.7-1.0s: Silence long enough, finalize turn                        │
│  │   ├── Mark current turn as complete                                    │
│  │   ├── Activate merge window (300ms grace period)                       │
│  │   └── If speech resumes within 300ms → MERGE into current turn        │
│  │       Else if silence continues > 300ms → NEW turn                    │
│  │                                                                          │
│  └── > 1.0s: Force finalize (speech definitely over)                      │
│      ├── Mark turn complete                                               │
│      ├── Expire merge window                                              │
│      └── Next speech will create NEW turn (definitely)                    │
│                                                                              │
│  Result: Natural speech recovery without cutting off mid-thought          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘


┌─ LANGUAGE LOCK STATE MACHINE ─────────────────────────────────────────────┐
│                                                                              │
│  Idle State                                                                │
│  └── sessionStartLanguages == nil                                         │
│      └── isLanguageLocked == false                                        │
│          └── Language swap button: ENABLED                                │
│                                                                              │
│  User Taps Mic (startMic)                                                 │
│  └── appState.startSession()                                              │
│      └── sessionStartLanguages = (current input, current output)          │
│          └── isLanguageLocked == true                                     │
│              └── Language swap button: DISABLED                           │
│              └── "locked" badge appears (amber)                           │
│                                                                              │
│  User Attempts Swap During Session                                        │
│  └── Button disabled (no tap response)                                     │
│  └── Or if tapped: swapLanguages() checks lock                            │
│      └── if isLanguageLocked: return (don't swap)                        │
│      └── Show diagnostic: "Language locked during session"                │
│                                                                              │
│  User Stops Session (stop)                                                │
│  └── appState.endSession()                                                │
│      └── sessionStartLanguages = nil                                       │
│          └── isLanguageLocked == false                                    │
│              └── Language swap button: RE-ENABLED                         │
│              └── "locked" badge disappears                                │
│                                                                              │
│  Next Session                                                             │
│  └── Can start with different language settings                           │
│      └── Language lock prevents accidental swaps only within session      │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
